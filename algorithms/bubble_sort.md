# 冒泡排序 (Bubble Sort)

## 逻辑描述
冒泡排序是一种简单的排序算法。它的工作原理是重复地遍历要排序的数组，一次比较两个相邻的元素，如果它们的顺序错误就把它们交换过来。遍历数组的工作是重复地进行直到没有再需要交换，也就是说该数组已经排序完成。

想象一下，你有一排气泡，每个气泡有不同的重量。每次遍历时，重的气泡会“冒泡”到后面，轻的留在前面。经过多次遍历，所有气泡就按重量排序了。

## 示例代码
```c
#include <stdio.h>

void bubbleSort(int arr[], int n) {
    int i, j, temp;
    for (i = 0; i < n-1; i++) {
        for (j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // 交换 arr[j] 和 arr[j+1]
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    printf("排序后的数组: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    return 0;
}
```

## 解释说明
在这个例子中，我们有一个数组 {64, 34, 25, 12, 22, 11, 90}。冒泡排序会多次遍历数组，每次比较相邻的两个元素并交换它们的位置。经过排序后，数组变成 {11, 12, 22, 25, 34, 64, 90}。

时间复杂度是 O(n^2)，适合小数据集。对于大数组，有更好的排序算法。
