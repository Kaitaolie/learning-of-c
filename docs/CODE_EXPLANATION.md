# C 代码说明

## 1. 程序 1：简单的可被 6 整除数求和

功能：计算区间 [1, 100] 中所有能被 6 整除的整数之和。

关键点：
- 使用 `for` 循环遍历 1 到 100。
- 通过 `i % 6 == 0` 判断是否可被 6 整除。
- 符合条件则累加到 `sum`。
- 最后用 `printf` 输出结果。

用途：适合练习基础循环、条件判断和变量累加。

---

## 2. 程序 2：优化版本，支持自定义区间

功能：读取用户输入的区间 `[a, b]`，计算该区间内所有能被 6 整除的数的和。

关键点：
- 使用 `scanf` 获取用户输入的 `a` 和 `b`。
- 检查 `a > b` 时打印错误并退出。
- 通过表达式 `i = a + (6 - a % 6) % 6` 计算区间内第一个能被 6 整除的数。
- 使用步长 `6` 的 `for` 循环遍历所有满足条件的数，避免每次都执行余数判断。
- 最后输出区间和。

优化说明：
- 这段代码利用了“等差数列”的思想，使得循环次数最少。
- `i += 6` 直接跳过不满足条件的数，性能比简单遍历更好。

## 3. 程序 3：质数判断器

功能：判断一个整数是否为素数。

关键点：
- 读取用户输入的整数 `n`。
- 如果 `n < 2`，直接认为该值既不是素数也不是合数。
- 先判断偶数：如果 `n != 2` 且 `n % 2 == 0`，则不是素数。
- 使用 `for` 循环判断从 3 开始的奇数因子，循环条件为 `i * i <= n`。
- 如果发现 `n % i == 0`，说明存在因子则不是素数；否则是素数。

数学依据：
- 任何合数都至少有一个因子小于等于其平方根。
- 只检查奇数、只循环到 `sqrt(n)`，使判断更高效。

## 4. 程序 4：模块化项目结构示例

功能：演示如何把函数声明与实现分离到头文件和源文件中。

组成：
- `max.h`：函数原型声明，使用 `#ifndef`、`#define`、`#endif` 进行头文件保护。
- `max.c`：包含 `max.h`，实现 `findMaxNum` 函数，返回两个整数中的较大者。
- `test.c`：包含 `<stdio.h>` 和 `max.h`，在 `main` 中调用 `findMaxNum`，打印结果。

意义：
- 这是 C 代码组织的常见方式，便于分离接口与实现，支持多文件编译。
- 如果需要扩展功能，可以继续增加新的函数、源文件和头文件。
