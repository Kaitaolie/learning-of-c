### 🥈 第二名：素数判断算法

📍 **位置**：`examples/prime_number/prime_number.c`

**关键优化**：
```c
// 优化前：O(n)
for (i = 2; i < n; i++) {
    if (n % i == 0) {...}
}

// 优化后：O(√n)
for (i = 2; i * i <= n; i++) {
    if (n % i == 0) {...}
}
```

**学习价值**：理解**数学优化**在编程中的重要性。

## 🎓 学习过程中的关键认识

### 1. **从简到复**
从基础的 `sum` 程序开始，逐步升级到模块化的多文件项目。每一步都是对前面知识的应用与扩展。

### 2. **代码组织很重要**
好的代码结构（头文件、源文件分离）比代码本身更重要。这是实际工程的基础。

### 3. **算法思维**
- 排序算法的复杂度差异巨大（$O(n²)$ vs $O(n \log n)$）
- 小的优化（二分查找中检查到 $\sqrt{n}$）能显著提升性能
- 理解"为什么"比记住"是什么"更重要

### 4. **递归的力量与局限**
- 递归优雅地表达某些问题（阶乘、树遍历）
- 但无限递归会导致栈溢出
- 需要理解递归的成本（时间/空间复杂度）

### 5. **内存管理的复杂性**
- 指针需要小心使用（野指针、内存泄漏）
- 这也是为什么高级语言有自动垃圾回收的原因
